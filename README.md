# Lines Filling Animation - React TypeScript

React TypeScript компонент для создания анимации заполнения линий с пропорциями 9:16 (формат рилсов).

## Особенности

- ✅ Пропорции 9:16 (формат рилсов)
- ✅ Анимация появления линий снизу вверх
- ✅ Настраиваемое расстояние между линиями
- ✅ Подписи к каждому прямоугольнику
- ✅ TypeScript поддержка
- ✅ Адаптивный дизайн

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

## Использование

```tsx
import LinesFilling from './LinesFilling'

const App = () => {
	const labels = [
		['A1', 'B1', 'C1', 'D1', 'E1', 'F1', 'G1', 'H1', 'I1', 'J1', 'K1'],
		['A2', 'B2', 'C2', 'D2', 'E2', 'F2', 'G2', 'H2', 'I2', 'J2', 'K2']
		// ... еще строки
	]

	return <LinesFilling labels={labels} lineSpacing={15} />
}
```

## Пропсы

| Проп          | Тип          | Описание                                                                        | По умолчанию |
| ------------- | ------------ | ------------------------------------------------------------------------------- | ------------ |
| `labels`      | `string[][]` | Двумерный массив подписей. Каждый вложенный массив - это строка с 11 элементами | -            |
| `lineSpacing` | `number`     | Расстояние между соседними линиями в пикселях                                   | `10`         |

## Структура данных

- `labels` - двумерный массив строк
- Каждая строка должна содержать ровно 11 элементов
- Количество строк может быть любым

## Анимация

- Линии появляются снизу вверх с задержкой 0.1 секунды между строками
- Прямоугольники в каждой строке появляются с дополнительной задержкой 0.05 секунды
- Каждая линия располагается над предыдущей с учетом `lineSpacing`
