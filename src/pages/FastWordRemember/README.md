# Fast Word Remember Animation

Анимация для быстрого вспоминания слов с изображением мозга.

## Описание

Компонент создает анимацию, где:

1. Над головой плавно появляется русское слово
2. В области рта по буквам анимированно выводится английское слово
3. Анимация автоматически переключается между парами слов

## Использование

### Базовое использование

```tsx
import FastWordRemember from './pages/FastWordRemember'

const wordPairs = [
	{ russian: 'Дом', english: 'House' },
	{ russian: 'Кошка', english: 'Cat' }
	// ... другие пары слов
]

const russianWordPosition = {
	x: 50, // 50% от ширины экрана
	y: 25 // 25% от высоты экрана (над головой)
}

const englishWordPosition = {
	x: 50, // 50% от ширины экрана
	y: 75 // 75% от высоты экрана (в области рта)
}

;<FastWordRemember
	wordPairs={wordPairs}
	russianWordPosition={russianWordPosition}
	englishWordPosition={englishWordPosition}
	autoPlay={true}
	interval={4000}
/>
```

### Настройка координат

Координаты задаются в процентах от размера экрана (1080x1920):

- `x: 0` - левый край экрана
- `x: 50` - центр экрана по горизонтали
- `x: 100` - правый край экрана
- `y: 0` - верхний край экрана
- `y: 50` - центр экрана по вертикали
- `y: 100` - нижний край экрана

### Рекомендуемые позиции

**Для русского слова (над головой):**

```tsx
russianWordPosition = {
	x: 50, // центр
	y: 20 // над головой
}
```

**Для английского слова (в области рта):**

```tsx
englishWordPosition = {
	x: 50, // центр
	y: 75 // в области рта
}
```

### Параметры

- `wordPairs` - массив пар слов `{russian: string, english: string}`
- `russianWordPosition` - координаты русского слова
- `englishWordPosition` - координаты английского слова
- `autoPlay` - автоматическое воспроизведение (по умолчанию `true`)
- `interval` - интервал между словами в миллисекундах (по умолчанию `3000`)

### Управление

- Кнопки "Предыдущее" и "Следующее" для ручного переключения
- Счетчик текущего слова
- Автоматическое воспроизведение с настраиваемым интервалом

## Анимации

1. **Русское слово**: плавное появление с эффектом масштабирования и пульсирующим свечением
2. **Английское слово**: появление по буквам с задержкой 150ms между буквами
3. **Эффекты**: свечение, тени, градиенты для визуального выделения

## Размеры

Компонент оптимизирован для экрана с пропорциями 1080x1920 (вертикальная ориентация).
